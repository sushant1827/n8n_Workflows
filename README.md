## Inventory Agent

A simple Inventory Agent that retrieves or updates inventory data stored in a Google Sheet based on user input.

Nodes Used:

1. AI Agent

2. OpenAI Chat Model

3. Simple Memory

4. Get Inventory (Tools)

5. Update Inventory (Tools)
