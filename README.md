# n8n_Workflows


## Inventory Agent

A simple `Inventory Agent` that retrieves or updates inventory data stored in a Google Sheet based on user input.

Nodes used: 

AI Agent, OpenAI Chat Model, Simple Memory, Get Inventory (Tools), Update Inventory (Tools)
